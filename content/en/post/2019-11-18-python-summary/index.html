1.1 Pycharm cookbook

1.1.1 如何更改每个项目的解释器版本?

File -> Settings ->project -> Project Interpreter

然后选择即可.

1.1.2 创建文件

File -> new

1.1.3 运行代码

可以一行行的运行代码,修改默认快捷键位ctl+enter

1.1.4 修改默认的快捷键

在File ->keymap下

1.1.5 编码设置

需要将pycharm的所有编码设置为UTF-8.

  1. File -> Setting -> Editor -> File Encoding

同时在每个文件的开头添加:

#-*- coding: utf-8 -*
