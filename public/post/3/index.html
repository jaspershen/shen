将其定义为一种特殊的函数.

实际问题中如何确定概率

  1. 频率说: 一系列重复随机试验中,考察随机时间发生的频率.从样本信息推算整体信息.通过实验.

  2. 主观说(贝叶斯派):根据以往的资料或者经验,形成的关于随机事件发生可能性的印象.先验信息.不是通过做实验.

例如: 考察某地区男婴出生的比例:只能根据以往资料.

考察某人是嫌疑人的可能性:完全不能用频率(完全没有重复).

  1. 等可能说(也是贝叶斯提出的):基本事件没有任何偏好,等可能的.

例如:投硬币得到正面的概率.投色子.分子热运动,每个分子超各个方向的运动.

几何定义.

基本事件个数有限,等可能原理称为古典概率.

形形色色概率的共同点:

  • 概率非负.
  • 规范:
  • 可加:如果两个事件互斥,则其概率可加.

概率定义:

定义在S的事件族,A上的实数值函数P为一个概率,如果他满足:

  1. 非负: \(P(A) \geq 0\)

  2. 规范性: \(P(S) = 1\)

  3. 可列可加性.

推广出来的性质:

  1. 不可能时间概率为0.

  2. 有限可加性: 若有一系列互斥事件,则其所有事件的发生概率,等于每个事件概率之和.

  3. 单调性: 若\(A \subset B\), 则\(P(A) \leq P(B)\)

  4. A为事件,则\(P(A) \leq 1\)

  5. 加法公式

A,B为事件,则:

\[P(A\bigcup B) == P(A) + P(B) - P(A \bigcap B)\]
